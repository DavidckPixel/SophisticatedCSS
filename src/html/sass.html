Features

Variables

Think of variables as a way to store information that you want to reuse throughout your stylesheet. You can store things like colors, font stacks, or any CSS value you think you'll want to reuse. Sass uses the $ symbol to make something a variable. Here's an example:

CSS


body {
  font: 100% Helvetica, sans-serif;
  color: #333;
}

SASS


$font-stack: Helvetica, sans-serif;
$primary-color: #333;

body {
  font: 100% $font-stack;
  color: $primary-color;
}

Nesting

When writing HTML you've probably noticed that it has a clear nested and visual hierarchy. CSS, on the other hand, doesn't.

Sass will let you nest your CSS selectors in a way that follows the same visual hierarchy of your HTML.

CSS


nav ul {
  margin: 0;
  padding: 0;
  list-style: none;
}
nav li {
  display: inline-block;
}

SASS


nav {
 ul {
  margin: 0;
  padding: 0;
  list-style: none;
 }
 li { display: inline-block; }
}

Operators

Doing math in your CSS is very helpful. Sass has a handful of standard math operators like +, -, *, /, and %.

In our example we're going to do some simple math to calculate widths for an aside & article.

CSS


article[role="main"] {
 float: left;
 width: 62.5%;
}

SASS


article[role="main"] {
 float: left;
 width: 600px / 960px * 100%;
}

Try this preprocessor

Sass can be used from the command line, or by using third party applications. You can download sass from the Github page or by running npm install -g sass in the commandline. Once u have it installed, you use it through the following steps:

  • Use sass input.scss output.css to convert a sass to a css file
  • Add the --watch flag to automaticly convert it
  • Folder paths can also be used to convert entire folders
