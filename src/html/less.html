Leaner Style Sheets

Less is a dynamic preprocessor style sheet languege that can be compiled into Cascading Style Sheets and run on the client-side or server-side. “Less” adds certain features and functionalities to CSS, such as variables, mixins, operations, and functions. These features let you develop an effective design layout that is both minimal and flexible. Moreover, the “Less” CSS preprocessor codes are compatible across a range of web browsers.

Features

Variables

It's not uncommon to see the same value repeated dozens if not hundreds of times across your stylesheets. Variables make your code easier to maintain by giving you a way to control those values from a single location. Less-variables are defined with @[name]

CSS


a,
.link {
 color: #428bca;
}

LESS


@link-color: #428bca;

a,
.link {
 color: @link-color;
}

Extending

Extend is a Less pseudo-class which merges the selector it is put on with ones that match what it references. A selector can contain multiple extend clauses, but all extends must be at the end of the selector.

CSS


nav ul {
 background: blue;
}
.inline,
nav ul {
 color: red;
}

LESS


nav ul {
  &:extend(.inline);
  background: blue;
}
.inline {
 color: red;
}

Merge

The merge feature allows for aggregating values from multiple properties into a comma or space separated list under a single property. merge is useful for properties such as background and transform.

CSS


.myclass {
 box-shadow: inset 0 0 10px #555, 0 0 20px black;
}

LESS


.mixin() {
 box-shadow+: inset 0 0 10px #555;
}
.myclass {
 .mixin();
 box-shadow+: 0 0 20px black;
}

Functions

Less also comes with an array of extra build in functions, such as If and Bool functions. But also some more complex math functions

CSS


div {
 margin: 0;
 color: black;
}

LESS


@some: foo;

div {
 margin: if((2 > 1), 0, 3px);
 color: if((iscolor(@some)), @some, black);
}

Try this preprocessor

Less can either be pre-compiled, or compiled in the browser. You can download sass from the Github page or by running npm install less -g in the commandline. Once u have it installed, you use it through the following ways:

  • Add [script src="less.js"] to HTML
  • Use lessc [flags] [source] [destination] to convert
  • Some flags are: --silent, --version, --no-color
