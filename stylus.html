Expressive, Dynamic, Robust CSS

Stylus was launched in 2010 by former Node JS developer T.J. Holowaychuck, nearly four years after the release of SASS and one year after the release of LESS. Stylus is built with Node JS and heavily influenced by the logical prowess of SASS and the simplicity of LESS. While Stylus is quite popular with Node JS developers, it never managed to take over a part of the market as large as SASS or Less.

Features

Flexible syntax

Semi-colons, colons, and braces are optional

// CSS
body {
 font: 14px/1.5 Helvetica, arial, sans-serif;
}
body button,
body button.button,
body input[type='button'],
body input[type='submit'] {
 border-radius: 5px;
}

// Stylus
body
 font 14px/1.5 Helvetica, arial, sans-serif
 button
 button.button
 input[type='button']
 input[type='submit']
 border-radius 5px

Nesting

When writing HTML you've probably noticed that it has a clear nested and visual hierarchy. CSS, on the other hand, doesn't.

Stylus will let you nest your CSS selectors in a way that follows the same visual hierarchy of your HTML.

// CSS
body {
 font: 14px/1.5 Helvetica, arial, sans-serif;
}
body #logo {
 border-radius: 5px;
}

// Stylus
body {
 font: 14px/1.5 Helvetica, arial, sans-serif;
 #logo {
  border-radius: 5px;
 }
}

Mixins

Stylus mixins allow you to define reusable functionality by defining in-language functions which can be called from within blocks

// CSS
button {
 -webkit-border-radius: 5px;
 -moz-border-radius: 5px;
 border-radius: 5px;
}

// Stylus
border-radius(val)
 -webkit-border-radius: val
 -moz-border-radius: val
 border-radius: val

button {
 border-radius(5px);
}

Variables

Stylus variables behave as you would expect in any other language, and may optionally be prefixed by the “$” character:

// CSS
#prompt {
 position: absolute;
 top: 150px;
 left: 50%;
 width: 200px;
 margin-left: -100px;
}

// Stylus
#prompt
 position: absolute
 top: 150px
 left: 50%
 width: w = 200px
 margin-left: -(w / 2)

Functions

"Stylus functions may be defined in the same manner as mixins, however their usage differs as they return values.

// CSS
body {
 foo: 6;
}

// Stylus
sum(nums...)
 n = 0
 n += num for num in nums
body
 foo: sum(1, 2, 3)

Try this preprocessor

Stylus can be used from the command line, or by using third party applications. You can download Stylus from the Github page or by running npm install stylus -g in the command line. Once you have it installed, you use it through the following steps:

  • Use stylus -w style.styl -o style.css to convert a stylus file to a CSS file
  • Add the --watch flag to automaticly convert it
  • Folder paths can also be used to convert entire folders
